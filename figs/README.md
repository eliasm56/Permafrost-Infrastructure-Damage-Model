# Figure Generation

This directory contains scripts used to generate the figures presented in the manuscript:

**Refined modelling of Arctic circumpolar building stock increases estimated mid-century permafrost degradation damages.**

The scripts operate on the **model outputs generated by the damage modeling workflows** located in the root of this repository.

---

# Scripts

| Script | Description |
|---|---|
| `uncertainty_sensitivity_plots.py` | Generates plots illustrating Monte Carlo uncertainty propagation and sensitivity analysis |
| `damage_amplification_map.py` | Produces Arctic maps showing regional amplification of damages between exposure datasets |
| `decomposition_waterfall_plot.py` | Creates waterfall plots illustrating how progressive improvements in exposure data increase damage estimates |

---

# Required Inputs

The figure scripts require the **CSV outputs produced by the modeling workflows**:

```
damage_model.py
damage_model_decomposition.py
```

These model outputs are located in the repository directories:

```
outputs/
│
├── damage_results/
└── decomposition_results/
```

The figure scripts read the following output files produced by the models:

```
BC_risk_analysis_v3_uncert_netcdf__ALL_MC_country_draws.csv
BC_decomposition_v1__B*_MC_country_draws.csv
BC_decomposition_v1__B*_MC_region_draws.csv
```

These files contain Monte Carlo simulation outputs used to generate:

- uncertainty analysis figures
- sensitivity analysis plots
- exposure decomposition plots
- regional damage amplification maps

---

# Dependencies

The figure scripts require the following Python packages:

```
python >= 3.9
numpy
pandas
geopandas
matplotlib
mapclassify
shapely
```

Install dependencies using:

```
pip install numpy pandas geopandas matplotlib mapclassify shapely
```

---

# Running the Scripts

Run the scripts from the repository root or from within this directory.

```
python uncertainty_sensitivity_plots.py
```

```
python damage_amplification_map.py
```

```
python decomposition_waterfall_plot.py
```

Each script produces **publication-quality figures** used in the manuscript and supplementary materials.
